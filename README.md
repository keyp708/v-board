# V Board Sprint5

小学生バレーボール用のコート主役UI版。

## 反映内容

- コート比率を 9m × 18m に固定
- アタックラインをネットから3m位置に配置
- コートをホームとして表示
- シーン追加・名前変更・複製・削除
- シーンごとに選手位置・守備範囲・ボール位置を保存
- 選手ドラッグ
- ボールドラッグ・高さ調整
- 守備範囲は真円スタート
- ハンドルで横伸ばし・縦伸ばし・円拡大・回転
- 長押しで選手情報編集
- 右スワイプで選手一覧
- 左スワイプでチーム一覧
- 上から下スワイプでシーン一覧
